API Documentation

API url: http://localhost:8088

API available pages:

/add: adds the input value to the current value and stores the operation in the stack

requires: an input value and an operation, either + or -

returns: a JSON object with the form {value: "value"}

/subtract: subtracts a value from the current value and stores the operation in the stack

requires: an input value and an operation, either + or -

returns: a JSON object with the form {value: "value"}

/pop: pops off the last operation performed by the stack and changes the current value to match

requires: no input

returns: a JSON object with the form {value: "value"} if the operation succeeds, if the stack is empty returns a 405 status code

/reset: clears the stack and resets the current value to "0"

requires: no input

returns: a status of 200 if the reset succeeds

/history: returns the current stack of operations as a JSON object

requires: no input

returns: a JSON object with the form {stack: "stack"}
